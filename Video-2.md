## What is .NET?

**.NET** is a free, open-source, cross-platform developer framework created by Microsoft for building various types of applications.

To build applications using .NET, Microsoft provides 3 things in .NET 

1. programming languages (C#, F#, VB.NET) 
                C# - most popular language for .NET
                F# - functional programming language
                VB.NET - similar to classic Visual Basic
2. Technologies (ADO.NET, ASP.NET, Entity Framework, etc.)
                ADO.NET - for database connectivity
                ASP.NET - for web development

3. Servers (IIS, Kestrel, etc.)
                IIS - Internet Information Services, a web server for hosting web applications
                Windows Server OS 
                SQL Server
