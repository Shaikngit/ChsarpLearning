# .NET Core Full Stack

- Every application will generally have 3 parts in it, those are:

1. UI (User Interface)
2. Middleware (Business Logic)
3. Database

Usually, the UI part is called Front End, the Middleware part is called Back End, and the Database part is called Database.

Here are the popular technologies used for each part:

## Full Stack Development Requirements

1. **Front End Technologies:** HTML, CSS, JavaScript, Angular or React
2. **Programming Language:** C#, Java, Python, etc.
3. **Database:** SQL Server, Oracle, MySQL, MongoDB, etc.
4. **Server-Side Technologies:** ASP.NET, Spring, Django, etc.

Currently we have three popular full stack development options: 

## .NET Full Stack Development
- **Programming Language:** C#
- **Database:** SQL Server
- **Server-Side Technologies:** ASP.NET

## Java Full Stack Development
- **Programming Language:** Java
- **Database:** Oracle
- **Server-Side Technologies:** Spring

## Python Full Stack Development
- **Programming Language:** Python
- **Database:** MySQL or MongoDB
- **Server-Side Technologies:** Django
